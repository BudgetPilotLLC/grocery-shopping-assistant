# Product Names

Current naming direction:

- Grocery app: `PantryCents`
- Budgeting app: `FlowMint`
- Company/account umbrella: `BudgetPilotLLC`, with `BudgetPilot Labs LLC` as the likely LLC name if formed.

## Why These Names

`PantryCents` is grocery-specific and points directly at the value proposition: planning pantry and grocery purchases with price awareness.

`FlowMint` is broader and more modern for personal finance. It suggests cashflow, clean systems, and a polished money workflow without colliding as directly with existing `BudgetPilot` finance apps.
