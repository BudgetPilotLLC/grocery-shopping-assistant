Grocery Shopping Assistant

A household grocery planner that turns a synced shopping list into store-by-store basket estimates, price-per-serving comparisons, and clearer shopping decisions.

See What It Does Starter Picks GitHub Profile

The project is being built for everyday grocery planning: make a list once, compare realistic basket options, and understand whether bulk sizes or nearby stores actually save money after servings are considered.

What It Does

Compares groceries in the way families actually shop.

Synced List Input

Reads a grocery list from a OneDrive-friendly text file today, with a mobile app planned for a smoother list experience later.

Basket Comparison

Compares item matches across configured stores including Amazon, Amazon Fresh, Publix, Aldi, and Sam's Club when data is available.

Price Per Serving

Normalizes package sizes and servings so bulk purchases can be compared against smaller packages without guesswork.

Mobile Roadmap

The planned mobile app will focus on list management, household shopping workflows, and reviewable recommendations before checkout.

Pricing Data

Transparent by design.

The current prototype uses a maintained catalog file and is being prepared for approved data providers. Amazon and Amazon Fresh are treated as separate sources because regular Amazon.com grocery listings and Fresh grocery offers can differ by availability, region, and delivery context.

When Amazon affiliate links or product content are used, this site may earn from qualifying purchases as an Amazon Associate. The app does not ask visitors for Amazon account passwords.

Current Sources

  • Local grocery catalog import
  • OneDrive-synced shopping list
  • Planned credentialed provider integrations
  • Planned mobile app list experience

Status

In active prototype development.

Grocery Shopping Assistant is not a store, delivery service, or checkout provider. It is a planning tool that helps compare available grocery options before a household decides where to shop. The public site also includes a small starter picks page for budget-conscious grocery planning supplies.
